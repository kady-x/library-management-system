# Library project

## This is a Data Structure Project Developed By AIU students group using Java Spring Boot and HTML

to run the code go to `LibraryApplication.java` and run it

if you faced any kind of environment problems and wanted to use docker build the image and use it

this is a build command example:
`docker build -t library-app .`

this is a run command example:
`docker run -p 8080:8080 --name library-app library-app:latest`
